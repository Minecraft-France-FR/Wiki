# Optimisation des villageois

- Pour optimiser un villageois (pour vos villageois de commerce), vous devez le renommer en `Opti` ou `NoAI`
  - Les majuscules comptent
  - Le name tag n'est pas consommé
- Vous ne pouvez pas échanger avec un villageois qui n'est pas optimisé
  - Vous pouvez tout de même voir leur échanges sans qu'ils soient optimisés
- Pour retirer l'optimisation d'un villageois, renommer-le en autre chose que `Opti` ou `NoAI`.
