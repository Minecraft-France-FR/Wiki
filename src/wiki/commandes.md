# Commandes

## 💸 Économie
- `/balance` : connaître son montant d'argent
- `/balancetop` : voir le classement des joueurs selon l'argent
- `/pay <joueur> <montant>` : donner de l'argent à un joueur

## ✉️ Social
- `/msg <joueur> <message>` : envoyer un message à un joueur
- `/r <message>` : répondre au message d'un joueur

## 🏡 Maisons (homes)
- `/delhome <nom>` : supprimer une maison
- `/home <nom>` : se téléporter à une maison
- `/renamehome <nom>` : renommer une maison
- `/sethome <nom>` : définir une maison
- `/homes` : voir ses homes dans le tchat
- `/maisons` : voir ses homes dans un menu

## 🔧 Utiles
- `/automessages` ou `/autom` : désactiver temporairement les messages automatiques
- `/menu` : afficher le menu principal
- `/help` : voir les commandes principales
- `/board` : activer ou désactiver le scoreboard
- `/helpop <message>` : envoyer un message aux modérateurs connectés

## 🧭 Téléportation dans les mondes
- `/spawn` : se téléporter au spawn
- `/ressource` : se téléporter dans le monde ressource
- `/enfers` : se téléporter dans le monde ressource du nether
- `/limbes` : se téléporter dans le monde ressource de l'end
- `/mondes` : voir les mondes et se téléporter via le menu
- `/back` : permet de se téléporter à sa précédente position avant une téléportation (ne fonctionne pas quand on est mort)
- `/tpr` : se téléporter aléatoirement dans un monde ressource
- `/warp <nom>` : se téléporter à un point de téléportation
- `/warps` : voir les points de téléportation disponibles

## 👁️ Téléportation entre joueurs
- `/tpa <joueur>` : demander à un joueur de se téléporter à lui
- `/tpaccept` : accepter la requête d'un joueur
- `/tpahere <joueur>` : demander à un joueur de se téléporter sur soi
- `/tpdeny` : refuser la requête d'un joueur
